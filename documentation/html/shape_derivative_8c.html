void freeMatrixMemory ( double **  pOverlapMatrix,
double **  pEigenvectors,
double **  pEigenvalues 
)

It frees the memory allocated with calloc/malloc/realloc for the vector structures defined in the main function and pointed here by pOverlapMatrix, pEigenvectors, and pEigenvalues.

Parameters
[in]pOverlapMatrixIt is a pointer to an vector structure (double*) storing the coefficients of the overlap matrix and that is intended to be freed up.
[in]pEigenvectorsIt is a pointer to an vector structure (double*) storing the eigenvectors of the overlap matrix and that is intended to be freed up.
[in]pEigenvaluesIt is a pointer to an vector structure (double*) storing the eigenvalues of the overlap matrix and that is intended to be freed up.

All pointers that have been freed up are also set to NULL.
