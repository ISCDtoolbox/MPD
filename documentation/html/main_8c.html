Main function of the MPD program.

Parameters
[in]argcthe (positive) number of arguments entered in the command line for executing the MPD program. Here, we must have argc=2, i.e. the command line must specify one input argument: the *.info (data) file used to load the parameters. Otherwise (zero or more than one argument) an error is returned by the MPD program.
[in]argvan array of pointers (char*) that points to the strings storing the names of the different arguments entered in the command line for executing the MPD program. Here, argv[0] always points to the name of the MPD program, argv[1] must point to the required name of the *.info (data) file, and argv[2]=argv[argc] must point to NULL; otherwise an error is returned by the MPD program.
Returns
It returns EXIT_FAILURE if an error is encountered, otherwise EXIT_SUCCESS is returned.

The function main performs on a given initial domain in a computational box (mesh variable) (parameter.)iter_max deformations to maximize the probability to find exactly (parameter.)nu_electrons in a chemicalSystem structure. The command line must specify the *.info (data) file, which must contain at least (minimal configuration) the *.wfn/ *.txt (chemical data) file, the (positive) number of electrons to look for, and optionally the *.mesh/ *.cube (mesh data) file to start with (if not specify a cube or a sphere is built).
