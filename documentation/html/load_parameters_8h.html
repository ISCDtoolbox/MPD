Detailed Description

Macros and prototypes related to the initial loading of parameters in the MPD algorithm.

Author
Jeremy DALPHIN
Version
1.0
Date
May 18th, 2018

This file contains the description of the all preprocessor constant and non-static function prototypes that are used to initially load the parameters of the MPD algorithm.

#define SAVE_WHERE   7

#define TRICK_MATRIX   1
